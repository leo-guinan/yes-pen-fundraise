# Artifact Review Protocol

## How Maintainers Review Contributions

### Review Criteria

**High-quality artifacts:**

- Are original and useful
- Are well-designed and clear
- Add real value to the Yes Room
- Can be referenced by others
- Stand the test of time

**High-quality evaluations:**

- Cover all 5 axes thoroughly
- Include specific examples and evidence
- Ask thoughtful questions
- Provide actionable suggestions
- Are well-written and clear

### Review Process

1. **Initial review** — Check for completeness and quality
2. **Community feedback** — Allow peer review and upvotes
3. **Founder endorsement** — Founder can endorse exceptional contributions
4. **Merge decision** — High-quality contributions are merged
5. **Points awarded** — Points accrue based on quality and type

### Scoring Guidelines

- **Minor artifact** — +10 pts
- **Substantial artifact** — +25 pts
- **Keystone artifact** — +50 pts
- **Community upvote** — +5 pts per upvote
- **Founder endorsement** — +20 pts

### Rejection Criteria

Artifacts/evaluations may be rejected if they:

- Are low-effort or spam
- Violate Code of Conduct
- Don't add value
- Are duplicates of existing content
- Contain misinformation

---

**Review maintains quality while encouraging contribution. Quality over quantity.**
