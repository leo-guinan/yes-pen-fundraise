# Evaluations

This folder contains scout evaluations. Each evaluation is a separate file.

## Naming Convention

Evaluations use the format: `YYYY-MM-DD-scout-handle-eval-v1.md`

Example: `2025-11-24-alice-scout-eval-v1.md`

## How to Submit

1. Use the [evaluation template](../templates/evaluation-template.md)
2. Follow the naming convention
3. Submit a PR
4. Wait for review and merge
